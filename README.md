# shennong-health-protocol
# 神农健康协议工具集
为程序员与矿工群体打造的开源健康管理方案，核心功能包括「神农转腰双效操」打卡、16+8 轻断食记录、血压数据追踪，帮大家改善久坐与高危工作带来的亚健康问题。
核心功能
转腰操打卡：动作识别 + 数据记录，生成每日健康报告
轻断食管理：自定义进食窗口，提醒喝水并记录饮食
血压追踪：对接智能设备，生成周 / 月血压变化曲线
快速开始
- 克隆仓库：git clone https://github.com/
shennong-protocol/shennong-health-protocol.git
- 安装依赖：npm install
- 启动服务：npm run dev
